
# **📘 Rechtliche Dokumente – OnlyRent by a_way**

In diesem Repository werden alle rechtlich relevanten Dokumente für
**OnlyRent** bereitgestellt – ein Produkt von **a_way**, der Initiative für automatisierte Immobilienprozesse.
Dazu gehören insbesondere:

* **Datenschutzerklärungen**
* **Impressumsangaben**
* **Agent- bzw. KI-Agent-spezifische rechtliche Hinweise**
* **rechtliche Rahmeninformationen zu automatisierten Immobilienprozessen**

Alle Dokumente werden in Markdown-Struktur abgelegt, um:

* Änderungen transparent nachverfolgen zu können
* Versionen sauber dokumentiert zu halten
* einheitliche rechtliche Grundlagen für alle OnlyRent-Prozesse und Tools bereitzustellen

---

## **📂 Inhalt**

Dieses Repository enthält u. a.:

* `/datenschutz/` → Datenschutzerklärungen für verschiedene Anwendungsbereiche
* `/impressum/` → Impressumsangaben für Tools, Webseiten, Agenten
* `/agents/` → Rechtliche Hinweise und Datenschutzinfos für KI-basierte Agenten und automatisierte Lösungen
* `/templates/` → Mustertexte (DSGVO-konform)

---

## **📄 Verwendung**

Die Dateien können:

* direkt auf Webseiten oder Plattformen eingebunden werden
* in Agenten, Chatbots oder automatisierten Workflows referenziert werden
* als Vorlage zur Weiterentwicklung rechtlicher Dokumente genutzt werden

Bitte denkt daran:
**Die Verantwortung für die finale rechtliche Prüfung bleibt beim betreibenden Unternehmen.**

---

## **🔐 Datenschutz-Hinweis**

Dieses Repository enthält **keine personenbezogenen Daten**.
Alle Dokumente sind rein informatorisch und dienen der rechtlichen Transparenz.
